Easy Prayer is a progressive web application. You can read on web page and install on your mobile device.

You can see how the times calculated on help page.

Features

  • Easy to use
  • Exact prayer times
  • Runs on web and mobile (Responsive)
  • Installable
  • Runs offline
  • Lightweight (About 100KB)
  • Lightning fast
  • Auto Location
  • Country and City selection
  • Calculation method selection
  • Asr selection
  • Monthly and weekly time table
  • Multi language
  • Printable (Press Ctrl+P)

Icons

  • o Location Settings
  • y Program info popup
  • 30 Monthly time table
  • 7 Weekly time table
  • b Delete Bookmark
  • s Settings (Select method, asr, language)
